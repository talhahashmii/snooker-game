# Snooker Game

A fully playable snooker simulator built with p5.js, featuring realistic ball physics, cushion collisions, cue control with adjustable spin, multiple game modes, and a custom procedural sound system.

## Features

- Realistic ball-to-ball and ball-to-cushion physics
- Cue control with adjustable spin and shot power
- Multiple game modes
- Custom sound generation system
- Smooth animations for ball movement and potting

## Tech Stack

- JavaScript
- p5.js
- HTML/CSS

## Project Background

This was built as a group project, with contributions across the physics engine, game logic, and core mechanics from both team members.

## Running Locally

Clone the repo and open `index.html` in a browser, or serve it with a local server (e.g. VS Code Live Server extension).

```bash
git clone https://github.com/talhahashmii/snooker-game.git
cd snooker-game
```

Then open `index.html` directly, or run a local dev server.
